StdUseCase.cpp File Reference

Detailed Description

Contains the base code required to make MSP430_Flasher.cpp operational. It uses MSP430.dll API calls to perform the desired operations and communicate with the MSP430.

Execute the program in the target memory.

If one or more breakpoints are set, StdUseCase_RunProgram will run the program, reading and displaying the CPU registers each time a breakpoint is hit.

Once a breakpoint is hit, it is cleared.

unsigned int inst_state

This bitmask contains global states and is accessed in various functions.

LSB
0 progress_bar in progress (log purposes)

1 write to external memory (for L092)

2 L092 selected? (for L092)

3 quiet mode selected?

4 alternative exit (true when error is encountered)

5 BSL write/erase unlock

6 log disabled

7 unlock INFO A memory

8 blow the security fuse

9 force FET firmware update

10 hardware breakpoint limit exceeded

11 default options used
